## Overview

Maya Professional Suite is a complete 3D production platform with rendering, pipeline tools, and scripting for studio-scale production.

Arnold rendering provides production-quality ray-traced rendering with physically based shading, volumetrics, and subsurface scattering. The IPR preview enables interactive look development with immediate feedback.

MEL and Python scripting provide complete programmatic control for pipeline integration, custom tools, and batch processing. The scripting environment enables studios to build custom workflows that integrate Maya into their production pipeline.

Alembic, USD, and FBX exchange handle data interchange with other DCC applications and game engines, ensuring that Maya integrates with any production pipeline.

Maya Professional Suite is the 3D production platform for studios that need comprehensive animation, VFX, and modeling capability with the pipeline integration that professional entertainment production demands.

## Performance & Integration

Maya is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while Maya processes operations quietly behind the scenes.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
